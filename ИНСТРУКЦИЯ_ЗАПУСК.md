# Инструкция по запуску папки build

## Что нужно для запуска

В папке `build` должны быть:

- Все файлы приложения (HTML, CSS, JS)
- Файл `launch.exe` (Windows) или `server` (Mac) - исполняемый файл сервера
- Папка `data/` (для хранения данных, создается автоматически)

## Первая установка (только один раз)

1. **Установите Node.js** (версия 12+)

   - Скачать можно с официального сайта: https://nodejs.org/
   - После установки перезапустите компьютер

2. **Установите зависимости проекта:**
   ```bash
   npm install
   ```

> **Примечание:** Node.js нужен только для сборки проекта. После сборки в папке `build` будет исполняемый файл сервера, который работает без установки зависимостей.

---

## Сборка проекта

### Для Windows:

```bash
npm run build:win
```

Создаст файл `build/launch.exe`

### Для Mac:

```bash
npm run build:mac
```

Создаст файл `build/server`

### Только фронтенд (без сервера):

```bash
npm run build
```

---

## Запуск приложения

Просто запустите исполняемый файл:

**Windows:** Дважды кликните на файл `build/launch.exe`

**Mac:** Дважды кликните на файл `build/server` или запустите из терминала:

```bash
./build/server
```

Приложение автоматически:

- Запустит сервер на порту 3001
- **Windows:** Откроет браузер (Chrome или Edge) в полноэкранном kiosk режиме
- **Mac:** Покажет URL в консоли (автоматическое открытие браузера работает только на Windows)
- Создаст папку `data/` для хранения данных (если её нет)
