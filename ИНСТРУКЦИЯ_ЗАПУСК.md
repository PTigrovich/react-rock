# Инструкция по запуску папки build

## Что нужно установить?

**Node.js** (версия 12+) - скачать можно с официального сайта: https://nodejs.org/

---

## Вариант 1: С установкой serve

1. Установите пакет `serve` (один раз):

   ```bash
   npm install -g serve
   ```

2. Перейдите в папку проекта и запустите:

   ```bash
   serve -s build
   ```

3. Откройте адрес, который покажет команда (обычно `http://localhost:3000`)
